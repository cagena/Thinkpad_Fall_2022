Assignment: Statistically Significant

In this homework assignment, you will practice creating and using a class. This assignment is to be completed individually. As it is homework, you may work with classmates, but you must do your own work; no two students should submit the same file.

The Assignment

Create a module with files stat_tracker.h and stat_tracker.cpp which contains a class which is given measurements and computes some statistical parameters from those measurements. The class should be named StatTracker and it should contain the following methods:

Method Purpose
StatTracker (void) Constructor initializes variables for zero data
void add_data (float) Add one data point to the set being tracked
void add_data (int32_t) Add a signed integer point to the data set
void add_data (uint32_t) Add an unsigned integer point to the data set
uint32_t num_points (void) Returns the number of data points added
float average (void) Returns the average of the data
float std_dev (void) Returns the standard deviation of the data
void clear (void) Deletes all the data, returning to no points added state

Your files, your class, and each function should have Doxygen style comments. These comments don’t need to be especially long and detailed, but they should contain @param and @returns sections as appropriate.
